Instruction of launch

1. You have to launch cmd and navigate to the folder where the project is located.
2. Next you have to execute the command "docker-compose up".
3. After that, you have to lauch TestTask.sln and then type the command "Update-Database" in Package Manager Console.
4. Finally you can launch the application by typing "https://localhost:33965/swagger/index.html" in the URL address of your web browser.
